What is Nexus? Chinese version is below.
Nexus is a collection of interactive globe products. Each product shares the same structure but explores a different topic. All products live in one GitHub repo, each in its own folder.
Repo: https://github.com/Chen0126Y/nexus
Live: https://chen0126y.github.io/nexus/<FolderName>/

How Every Product Works
All products use the same 3-layer navigation:
🌍 Globe
      ↓ click a country
✨ Nebula  (node network — people, places, events…)
      ↓ click a node
🃏 Card  (detail info + external links)
What changes between products:

The data (people, ruins, news…)
The colors and music theme
The fields shown on the detail card

What never changes:

The globe engine
The nebula physics animation
The audio system
The EN / 中 language toggle


Adding a New Product

Create a new folder in the repo — e.g. News/ or Film/
Copy the index.html from the most similar existing product
Replace the data — clear old content, fill in new nodes and relationships
Update colors, title, and loading text to match the new theme
Add one row to the root README.md with the product name and link


You do not need to touch the globe, nebula, or audio code.


Image Rules
Use real photos only — no AI-generated images.
Recommended sources (accessible in China and globally):

images.unsplash.com — append ?w=300&q=80 to the URL
upload.wikimedia.org — Wikipedia media

#Nexus — 使用说明
这是什么？
Nexus 是一组互动地球仪产品。每个产品结构相同，但主题不同。所有产品放在同一个 GitHub 仓库里，每个产品一个独立文件夹。
仓库地址： https://github.com/Chen0126Y/nexus
访问地址： https://chen0126y.github.io/nexus/<文件夹名>/

当前产品
文件夹产品名主题Business/NEXUS全球商业精英关系图Civilization/Echoes of Civilization世界历史文明探索

每个产品的结构
所有产品都是同一套三层交互：
🌍 地球仪
      ↓ 点击国家
✨ 星云图（节点网络——人物、遗迹、新闻……）
      ↓ 点击节点
🃏 详情卡（介绍 + 外部链接）
每个产品不同的地方：

数据内容（人物、遗迹、新闻……）
颜色和音乐风格
详情卡显示的字段

永远不变的部分：

地球仪引擎
星云物理动画
音频系统
中文 / EN 语言切换


如何新增一个产品

在仓库里新建一个文件夹，例如 News/ 或 Film/
复制最接近新主题的现有产品的 index.html 作为起点
替换数据——清空原有内容，填入新主题的节点和关系
修改颜色、标题和加载文案，匹配新主题的风格
在根目录 README.md 里加一行，注明新产品名称和访问链接


不需要改地球仪、星云、音频这三部分代码。


图片规则
只用真实照片，不用 AI 生成图。

images.unsplash.com — 网址末尾加上 ?w=300&q=80
upload.wikimedia.org — 维基媒体图库
